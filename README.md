# basic-pack
Folder structure with Vanilla everything and sass. 

Quick folder structure for smaple and not complex application
Used mostly for Static web apps/pages

The overall structure:

[--/dist/
  ---/css/
   ----main.css
   ----main.min.css
  ---/images/
  ---/scripts/
   ----main.js
  ---index.html
]

[--/sass/
  ---/core/
    ----/reset/
      ----_reset.scss
  ---/design/
    ----/layout/
      ----_0-section-1.scss
      ----_1-section-2.scss
      ----_2-section-3.scss
      ----_3-section-4.scss
      ----_4-section-5.scss
      ----_5-section-6.scss
    ----/variables/
      ----_variables.scss
   ---_design.scss
  --_main.scss
 ]
